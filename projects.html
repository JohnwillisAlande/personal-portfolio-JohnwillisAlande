My Projects

A showcase of my most impactful and innovative work

Afya Health

Afya Health is a comprehensive drug dispensing web application designed to streamline the process of managing prescriptions, inventory, and patient records for pharmacies and clinics. The platform features secure user authentication, real-time stock updates, and automated alerts for low inventory. Pharmacists can efficiently dispense medication, track patient history, and generate insightful reports. The intuitive interface ensures ease of use for both medical staff and patients, while robust backend integration with PHP and MySQL guarantees data security and reliability.

Tech: HTML, CSS, JavaScript, PHP, MySQL
Kadikazi

Kadikazi is a modern car wash management system that enables customers to book appointments, make secure payments, and receive real-time notifications. The application supports multiple service packages, tracks appointment history, and provides analytics for business owners. Built with ReactJS for a dynamic user experience, Kadikazi integrates seamlessly with PHP and MySQL for backend operations. The system prioritizes security, offering encrypted transactions and user data protection, while its responsive design ensures accessibility across devices.

Tech: HTML, CSS, ReactJS, PHP, MySQL
Flood Watch

Flood Watch is an advanced flood prediction and disaster management system leveraging machine learning to provide early warnings and actionable insights. The platform collects and analyzes environmental data, predicts flood risks, and visualizes affected areas using interactive maps. Emergency responders and local authorities can coordinate relief efforts, send alerts, and manage resources efficiently. Developed with Python, ReactJS, and Bootstrap, Flood Watch combines scientific accuracy with user-friendly dashboards, making it an essential tool for disaster preparedness and response.

Tech: HTML, Bootstrap, ReactJS, Python, MySQL
